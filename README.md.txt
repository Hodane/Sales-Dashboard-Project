# Sales Dashboard Project

## Project Overview
This is a beginner-friendly Business Analyst project demonstrating data analysis and dashboard creation using Excel. The project focuses on sales data analysis, monthly trends, and regional insights.


## Analysis
- Pivot tables created for:
  - Revenue by Product
  - Monthly Sales Trend
  - Revenue by Region
- Charts generated to visualize key trends.
- Insights written based on analysis.
